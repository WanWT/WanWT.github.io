Source：2013 ACM/ICPC Asia Regional Online —— Warmup2
Link: http://acm.hdu.edu.cn/showproblem.php?pid=4725

题意：

N 个节点，分别属于不同的集合，其中在集合X 内的点到集合 X+1(反之一样)的点的路径长度为 C ,另外还有 M条 额外的路，表示从 u 到 v 的双向路径长度为 w,求 1-n 的最短路。

范围：T <= 20 && 1 <= N,M <= 100000 && 1 <= C <= 1000 && 1 <= w <= 10000

思路：

主要还是建图,对于每个集合，我们要把集合X拆为两个点，可以理解为入点与出点，X内的点与入点连一条长度为 0 的单向边，入点与 X + 1 的出点连接一条 长度为 C 的单向边， X + 1的出点与X + 1集合内的点连一条单向边，反之，X + 1 内的点与 X + 1 的入点连一条边， X + 1 的入点与 X 的 出点连一条边，X 的出点与 X 内的点连一条边。这样就不会出现如果只拆一个点， X 内的相互之前会有长度为 C 的连边的情况。
