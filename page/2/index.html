2016 Asia Hong Kong Online Preliminary (2016亚洲区域赛香港站网络赛) - Problem C - Boxes

By WwTT0T

发表于 2016-09-11

Source：Hong Kong Regional Online Preliminary 2016
Link: https://open.kattis.com/problems/boxes

题意：

N个盒子，编号从 1-N ,盒子之间有嵌套关系，一个盒子可以套住多个盒子，被套住的盒子还能套盒子，最后盒子间的嵌套关系形成一棵树，给出 Q 个请求，每个请求有 M 个盒子编号，问这些盒子所嵌套的盒子数总共有多少个，包括它自身，每个盒子只能计算一次，如 1被2套住，2被3套住，请求是{2,3}，答案为3，总共 {1,2,3} 被套住

范围：1 <= N <= 200000 && 1 <= Q <= 100000 && 1 <= M <= 20
